GOPATH

GOPATH 为 GO 的一个重要的环境变量。

GOPATH 作为 import 包时的搜索路径，同时也是项目源代码和二进制文件的目录，go get、go install 和 GO 的一些其他工具都会用到 GOPATH。

GOPATH 设置

GOPATH 为一个路径，支持多个，因此也可以说 GOPATH 是一个路径列表。

当有多个 GOPATH 路径时，需要用分隔符分隔不同的路径：

  • Windows 分隔符为 ; 
  GOPATH=X:\Go\A;X:\Go\B
  • 类 Unix 系统和 Mac 为 : 
  export GOPATH=$HOME/Go/A:$HOME/Go/B

( 注: 当有多个 GOPATH 路径时，go get 获取的包将默认存放在第一个路径目录下。 )

目录说明

GOPATH 目录一般会有三个子目录。

  • bin 经过编译的可执行文件目录
  • pkg 编译时生成的中间文件( 如：.a 文件 )
  • src 用于存放项目源代码( 如：.go .h .c 等源码 )
